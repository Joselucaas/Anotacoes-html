Protocolos

Protocolos são um conjunto de regras para a comunicação em rede. Eles determinam como computadores se comunicam: como iniciar, como manter e como encerrar.

HTTP - Hyper Text Transfer Protocol (HTTP) é o protocolo usado pelos servidores web para trasnmitir documentos HTML pela internet.

FTP - File Transfer Protocol (FTP) é o outro protocolo muito utilizado para transmitir arquivos, porém sua interface gráfica nao é tão agradável como o HTTP.

TCP - Transfer Control Protocol (TCP) é um conjunto de protocolos para transmissão pela web, com função de verificar se os dados são transferidos de forma correta, na sequência apropriada e sem erros.

IP - Internet Protocol (IP) é um protocolo de endereçamento, que fornece o endereço dos computadores na rede.

URL (Universal Resource Locator) é o sistema de endereçamento usado pelos navegadores para localizar redes, computadores e arquivos na internet.

Listas

<ol> </ol> - São para criar listas ordenadas (1, 2, 3...) ou (A, B, C...), use essa tag para formas listas que precisam de uma ordem, como receitas ou tutorias. E para cada item da lista temos a tag <li> </li>

<ul> </ul> - São para criar listas não ordenadas com pontinhos, quadradinhos ou circulos. Use essa tag para formar listas que não precisa de uma ordem, como lista de material escolar. E também é usado a tag <li> </li> para cada item da lista.

O atributo type serve para especificar o tipo de marcador que irá aparecer na lista.

LISTA ORDENADA

  • A- Para exibir letras maiúsculas
  • a - Para exibir letras minúsculas
  • I - Para exibir números romanos grandes
  • i - Para exibir números romanos pequenos.
Sintaxe: <ol type="estilo do número/letra">

LISTA NÃO ORDENADA

  • disc - Para exibir pontos
  • circle - Para exibir circulos
  • square - Para exibir quadrados

Sintaxe: <ul type="estilo do marcador">
É possível juntar as duas listas!!

Tipos de Imagens

JPEG ou JPG - É o método mais utilizado para comprimir imagens fotográficas. Suporta até 16,8 milhões de cores e não permite fundos transparentes.

GIF - Seu formato suporta 8 bits por píxel, permitindo o uso de uma paleta de até 256 cores. É mais indicado para gráficos, ícones e imagens que não necessitam de muitas cores. Permite imagens com fundo transparente.

PNG - É o subustituto do formato GIF. Assim como o JPG, permite 16,8 milhões de cores e, como o GIF, permite fundos transparentes

Para inserir imagens em arquivos HTML temos que inserir a tag <img> e o atributo src que significa source(fonte)
Sintaxe: <img src="pasta/nome imagem">

O atributo alt permite vincular um texto explicativo à imagem.
Sintaxe: <img alt="Texto Explicativo">

Criando Links

a tag anchor <a> </a> é utilizada para criar links no HTML.

Precisamos do atributo href para especificar a URL do documento linkado ou interligado
Sintaxe: <a href="URL"> Texto Descritivo </a>

É possível linkar o site com o email.

Sintaxe:
<a href='mailto:lucas.chag0608@gmail.com'> Entre em contato </a>

Tem como criar um link para uma seção na mesma página

Basta colocar a tag <a name='nomeDaSeção'> na parte (seção) do site em que você deseja linkar.
Depois coloca a tag < a href='#nomeDaSeção'>Texto que será linkado</a>

Imagens Mapeadas

Imagem mapeada é uma imagem dividida em diferentes áreas e cada área é interligada a um documento.
Uma imagem mapeada é diferente de um link em uma imagem.

Há dois tipos de imagens mapeadas

Server-side (no servidor) - Na imagem mapeada server-side, você define um arquivo especial chamado arquivo de definição das coordenadas com todos os detalhes da imagem mapeada e armazena este arquivo no servidor.

Client-side (no cliente) - Na imagem mapeada client-side, você inclui todos os detalhes da imagem mapeada no próprio documento.

Criando e Formatando Tabelas

Para criar uma tabela é necessario três tags:

<table> - </table>: Reponsável por criar a tabela.

<td> - </td>: Responsável por criar as células/colunas.

<tr>: Responsável por criar as linhas.

A tag <table> possui vários atributos que permitem, entre outros, controlar:
- O alinhamento da tabela na página.
- O alinhamento dos dados nas células.
- A largura e cor da borda. <table border='nº de pixels'> / <table bordercolor = 'cor' >
- As dimensões da tabela. width - define largura / height - define altura

O atributo <cellspacing> especifica o espaço entre a borda da tabela e as células.

O atributo <cellpadding> especifica o espaço entre os dados e a borda de uma célula.

Atributos da tag <td>

O atributo <valing> serve para alinhamento vertical: top, bottom e center.

O atributo <colspan> é utilizado para mesclar uma célula em mais de uma coluna, de forma horizontal, e o valor é dado em números.

O atributo <rowspan> é utilizado para mesclar uma célula em mais de uma coluna, de forma vertical, e o valor é dado em números.

Criando e formatando formulários

Para criar um formulário em HTML tem que utilizar a tag <form> </form>

O primeiro atributo a usar na tag <form> é o action.
Ele vai especificar a URL do script que processará os dados do formulário.

Os dados podem ser enviados para um email através da sintaxe
<form action="mailto:seuemail@endereco.com">
Ou os dados podem ser processados por meio de um documento com uma determinada linguagem
<form action="processa.js">

O atributo <method> especifa a maneira de envio dos dados do formulário para o servidor web para que sejam processados.
Existem dois métodos que podem ser utilizados: POST e o GET

Post - É o mais utilizado, porque permite a transmissão de uma maior quantidade de dados ao servidor web
Get - O valor GET é utilizado para respostas únicas, como uma string de texto.

A sintaxe completa da tag <form> fica assim:
<form action="URL" method="get" ou "post">

Atributos da tag <input>

type = Especifica o tipo de controle a ser criado. Se nada for especificado, o padrão é "text"

value = Especifica o valor inicial do controle. É opcional, com excessão de quando o TYPE for "radio" ou "checkbox".
name = Nomeia cada campo do formulário, para que o script possa manipular os dados.
maxlenght = Quando o valor de TYPE for "text" ou "password", eles estabelecem o número máximo de caracteres do campo. O valor padrão é ilimitado.
src = Quando o valor TYPE for "image", este atributo especifica a localização da imagem usada para decorar o botão enviar.
checked = Quando TYPE tem o valor "radio" ou "checkbox", este atributo faz com que um dos botões já esteja selecionado
size = Especifica o tamanho inicial do controle. A largura é dada em pixel, com excessão de quando o valor for "text" ou "password".
Nestes casos, o valor refere-se ao número de caracteres.

Os possiveis valores para o atributo TYPE

image = Criar um botão de enviar baseado em uma imagem especifica, cuja URL deve ser fornecida.
file = Criar um controle para seleção de arquivo. O valor de atributo deve ser configurado como o nome do arquivo.
reset = Cria um botão e reset, que limpa todos os campos.
button = Cria um botão para ser pressionado. O atributo deve ter o nome o nome do botão.
submit = Cria um botão de enviar.
password = Cria um campo de texto sem exibir o que está sendo digitado.
radio = Cria um botão de escolha única.
text = Cria um campo de texto de uma linha.
checkbox = Cria caixa de seleção.
hidden = Cria controle oculto.

A tag <textarea> </textarea> é utilizada para adicionar uma área de texto maior em um formulário, para receber várias linhas de textos.

Os atributos dela são os "rows" que especifica o números de linhas(altura) e o "cols" que especifica o número de colunas(largura) da área de texto.

A tag <select> </select> é utilizada para criar uma lista de opções ou um menu.
Para criar uma lista basta pôr a tag e dentro dela colocar cada <option>Item1</option>




Para criar o menu tem que colocar o parâmetro "multiple" na tag <select>
E dentro da tag select colocar cada <option>Item1</option>
